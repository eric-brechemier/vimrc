## Setup ##

    $ cd ~
    $ wget https://github.com/eric-brechemier/vimrc/raw/master/.vimrc
    $ mkdir -p .vim/colors
    $ cd .vim/colors
    $ wget https://github.com/eric-brechemier/vimrc/raw/master/.vim/colors/norwaytoday.vim
    $ cd ~

Note:
The option --no-check-certificate may be needed for wget.
